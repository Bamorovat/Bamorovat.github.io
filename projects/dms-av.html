Project Overview

The Driver Monitoring System (DMS) for Autonomous Vehicles is a comprehensive safety solution developed at Conigital that combines advanced computer vision techniques with real-time processing to monitor driver behavior and alertness. This system represents a critical safety component for semi-autonomous and autonomous vehicles, ensuring driver readiness to take control when needed.

Development Context

This project was developed during my tenure as a Perception Engineer at Conigital, where I led the development of this comprehensive monitoring system. The project involved creating a complete end-to-end solution from algorithm development to deployment-ready code.

System Components

1. Eye Calibration Module

The eye calibration system performs initial setup to establish baseline eye characteristics for each individual driver. This personalization improves the accuracy of subsequent monitoring operations.

2. Blink Detection Algorithm

Advanced computer vision algorithms analyze eye region characteristics to detect blink patterns. The system can distinguish between normal blinks, prolonged closures, and microsleep episodes.

3. Yawn Detection System

Facial landmark detection and analysis algorithms monitor mouth region characteristics to identify yawning behavior, which is a key indicator of driver fatigue.

4. Head Orientation Tracking

3D head pose estimation provides real-time monitoring of driver head position and orientation, enabling detection of distraction events and attention focus.
