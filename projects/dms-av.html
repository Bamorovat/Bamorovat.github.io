Project Overview

ROS_DMS is a modular, real-time Driver Monitoring System for ROS, designed to detect drowsiness and attention using only a camera. It features robust eye calibration, blink/yawn detection, head pose estimation, and real-time alerts, making it ideal for autonomous and semi-autonomous vehicles.

The system processes camera images to extract facial landmarks, computes eye and mouth aspect ratios, and triggers alerts for drowsiness or inattention. All processing is optimized for real-time performance on typical robotics hardware.

Key Features

Eye Calibration

Automatic, per-driver calibration for robust thresholding.

Blink Detection

EAR-based blink and microsleep detection.

Yawn Detection

MAR-based yawn/fatigue detection.

Head Pose Estimation

3D head pose estimation for attention/distraction monitoring.

Real-time Processing

Optimized ROS nodes for interactive frame rates.

Alerts & Media

Audio alerts and configurable media files.

Capabilities

Personalized calibration for each driver
Robust blink and yawn detection
3D head-pose estimation for distraction/attention
Real-time audio/visual alerts
Modular ROS node design for easy integration
